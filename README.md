# bcg729-test-patterns
test patterns for bcg729 library

This repository independtly stores the test patterns for bcg729 repository. 

https://gitlab.linphone.org/BC/public/bcg729  ( github mirror : https://github.com/BelledonneCommunications/bcg729 )

So the main repository can be cloned quickly for users that do not want to run tests.

These patterns were generated mainly using the reference ITU implementation. They are directly accessed by the library when running tests, no need for manual download
